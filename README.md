# Snow Rush

Snow Rush is a production-ready Unity mobile endless runner scaffold with hyper-casual sled gameplay, low-poly snowy environments, offline-first saves, procedural generation, shops, missions, achievements, and Firebase-ready backend hooks.

## Highlights

- Mobile-first 3-lane sled controls: swipe left/right, jump, and slide.
- Endless procedural icy road with increasing speed and obstacle density.
- Obstacles: rocks, trees, barriers, cars, snowballs, and ice holes.
- Collectibles: coins, gems, magnet, shield, double coins, speed boost, and freeze time.
- Characters: Penguin, Polar Bear, Snow Kid, Santa, and unlockable skins.
- Environments: snow mountains, frozen city, ice forest, snow village, night mode, weather, and snowfall-ready particle folders.
- Economy: coin shop, character shop, skin shop, upgrade system, daily rewards, lucky spin, missions, achievements.
- UI: glassmorphism-ready panels, animated buttons, counters, pause, game-over, rewarded ads, and IAP extension points.
- Backend: Firebase Authentication, Firestore, Analytics, Crash Reporting, and Cloud Save facade.
- Optimization: object pooling, Addressables-ready folders, URP mobile pipeline, lightweight low-poly asset organization.

## Repository Layout

```text
Assets/SnowRush
├── Art              # Materials, models, textures, shaders, particles
├── Audio            # Music and sound effects
├── Prefabs          # Characters, obstacles, collectibles, power-ups, environment, UI
├── Scenes           # Boot, menu, and gameplay scenes
└── Scripts          # Clean feature-sliced C# runtime code
```
